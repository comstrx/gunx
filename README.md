# ✨ gunx

Integrated project management CLI for real-world development workflows.

`gunx` is a command-line tool for creating, managing, running, testing, building, deploying, and maintaining software projects across multiple stacks.

## Goal

Make project management fast, clean, and predictable from one unified command system.

Instead of remembering different commands for every framework, `gunx` gives you one simple workflow.

```txt
new → add → run → check → test → build → deploy
````

## Features

* Create new projects
* Detect project stack automatically
* Run development servers
* Run tests
* Run checks and linters
* Build projects
* Clean cache and generated files
* Manage Git and GitHub workflows
* Manage environment files and secrets
* Support deployment workflows
* Provide a fast hotpath CLI for daily development

## Example Usage

```bash
gunx new laravel api
gunx run
gunx check
gunx test
gunx build
gunx deploy
```
